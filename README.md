## 🛠️ 技术栈 (Tech Stack)

- **框架**: [Next.js 14](https://nextjs.org/) (App Router)
- **后台**: [Payload CMS](https://payloadcms.com/) (Headless / Type-safe)
- **核心**: [React](https://reactjs.org/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **交互**: 原生 CSS Scroll Snap + React Hooks 状态管理

## 📦 如何在本地运行 (Getting Started)

如果你想在本地跑起这个模板，只需简单的几步：

1. 克隆这个仓库到本地：
```bash
git clone https://github.com/Guanghao-Li/guanghaoli-blog
```

2. 进入项目目录并安装依赖：
```bash
cd 你的仓库名
npm install
```

3. 启动本地开发服务器：
```bash
npm run dev
```

4. 打开浏览器访问 [http://localhost:3000](http://localhost:3000) 即可预览。
